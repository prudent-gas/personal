Module idma_rt_midend_synth

Synth wrapper for the RT midend for the iDMA

Parameters

NumEvents: int unsigned

Number of parallel events

EventCntWidth: int unsigned

The width of the event counters (count multiple of clock period)

NumOutstanding: int unsigned

Number of outstanding DMA events

addr_t: type

Address type

idma_nd_req_t: type

ND iDMA request type

idma_rsp_t: type

iDMA response type

event_cnt_t: type

DO NOT OVERWRITE: Derived event counter type.

Ports

clk_i: input logic

Clock

rst_ni: input logic

Asynchronous reset, active low

event_counts_i: input event_cnt_t [NumEvents-1:0]

The threshold an event is triggered

src_addr_i: input addr_t [NumEvents-1:0]

The source address of the event

dst_addr_i: input addr_t [NumEvents-1:0]

The destination address of the event

length_i: input addr_t [NumEvents-1:0]

The length of the event

src_1d_stride_i: input addr_t [NumEvents-1:0]

The source stride of the event

dst_1d_stride_i: input addr_t [NumEvents-1:0]

The destination stride of the event

num_1d_reps_i: input addr_t [NumEvents-1:0]

The number of repetitions of the event

src_2d_stride_i: input addr_t [NumEvents-1:0]

The source stride of the event

dst_2d_stride_i: input addr_t [NumEvents-1:0]

The destination stride of the event

num_2d_reps_i: input addr_t [NumEvents-1:0]

The number of repetitions of the event

event_ena_i: input logic [NumEvents-1:0]

Enable the given event

event_counts_o: output event_cnt_t [NumEvents-1:0]

Current state of the counters (debugging)

nd_req_o: output idma_nd_req_t

ND iDMA request

nd_req_valid_o: output logic

ND iDMA request valid

nd_req_ready_i: input logic

ND iDMA request ready

burst_rsp_i: input idma_rsp_t

iDMA 1D response

iDMA 1D response valid

burst_rsp_ready_o: output logic

iDMA 1D response ready

Bypass: ND iDMA request

Bypass: ND iDMA request valid

nd_req_ready_o: output logic

Bypass: ND iDMA request ready

burst_rsp_o: output idma_rsp_t

Bypass: iDMA 1D response

burst_rsp_valid_o: output logic

Bypass: iDMA 1D response valid

burst_rsp_ready_i: input logic
