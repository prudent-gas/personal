Module idma_desc64_reg_wrapper

This module implements backpressure via ready/valid handshakes

for the regbus registers and exposes it to the descriptor fifo

Parameters

reg_rsp_t: type

Ports

clk_i: input logic

rst_ni: input logic

reg_req_i: input reg_req_t

reg_rsp_o: output reg_rsp_t

reg2hw_o: output idma_desc64_reg2hw_t

hw2reg_i: input idma_desc64_hw2reg_t

devmode_i: input logic

input_addr_valid_o: output logic

input_addr_ready_i: input logic

Signals

request: reg_req_t

response: reg_rsp_t
