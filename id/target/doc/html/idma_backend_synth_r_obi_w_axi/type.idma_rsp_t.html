Typedef idma_rsp_t

typedef struct packed {                                              
        logic         last;                                              
        logic         error;                                             
        err_payload_t pld;                                               
    } idma_rsp_t;

1D iDMA response type:

  • last: the response of the request that was marked with the opt.last flag

  • error: 1 if an error occurred

  • pld: the error payload
