Module idma_nd_counter

A simple counter needed for the n-dimensional midend.

Parameters

Width: int unsigned

The width of the counter

Ports

clk_i: input logic

Clock

rst_ni: input logic

Asynchronous reset, active low

en_i: input logic

Enable the counter

clear_i: input logic

Clear the counter

limit_i: input logic [Width-1:0]

The limit where the counter overflows

done_o: output logic

Overflow signal
