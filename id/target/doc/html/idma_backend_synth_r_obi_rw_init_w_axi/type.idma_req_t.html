Typedef idma_req_t

typedef struct packed {                                              
        tf_len_t   length;                                               
        addr_t src_addr;                                             
        addr_t dst_addr;                                             
        logic     user;                                                 
        options_t  opt;                                                  
    } idma_req_t;

1D iDMA request type:

  • length: the length of the transfer in bytes

  • *_addr: the source / target byte addresses of the transfer

  • opt: the options field
